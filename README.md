# Arquivos de linguagem em Português do Brasil para Laravel 5.6

## Instalação
1. Download
  ```
  $ cd resources/lang/
  $ git clone https://github.com/enniosousa/laravel-5.5-pt-BR-localization.git ./pt-BR
  ```
2. Configuração
  ```
  // Linha 81 do arquivo config/app.php
  'locale' => 'pt-BR',
  ```
